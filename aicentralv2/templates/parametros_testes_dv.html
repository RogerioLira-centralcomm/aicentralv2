DV360 · Hierarquia, autenticação e relatórios

Hierarquia: Campanha → Pedido de inserção (IO) → Line item. Orçamentos, pacing e grande parte do targeting aplicam-se ao IO/LI; a campanha agrupa objectivos.

Autenticação: Esta aplicação usa OAuth no .env (DV360_CLIENT_ID, DV360_CLIENT_SECRET, DV360_REFRESH_TOKEN, DV360_PARTNER_ID) para chamar displayvideo.googleapis.com.

Métricas de desempenho: Impressões, cliques e relatórios completos costumam vir do UI do DV360, Query Tool ou da Bid Manager API — produto e âmbitos OAuth diferentes da Display Video API v4.
